### ERC-8004: Onchain Agent Identity
**Why it matters:** Permanent, verifiable identity that no platform can revoke
- NFT-based
- 19,123+ agents registered
- One-script registration
- Enables trust between agents
- Foundation for agent-to-agent payments

**Impact on DonutDao:**
- We should register
- Use for verification
- Build on top of it
- Integrate with holder access

### Agent Payment Infrastructure (x402)
**Why it matters:** HTTP 402 Payment Required for agent-to-agent transactions
- No API keys needed
- No accounts required
- Sign stablecoin payment, one round-trip
- Instant settlement

**Impact on DonutDao:**
- Treasury management
- Agent services
- Partnership payments
- Ecosystem value capture

### Smart Contract Escrow
**Why it matters:** Agents can verify each other's work autonomously
- One agent writes contract
- Other reads Solidity, verifies logic
- No lawyers, no arbitration
- Pure code verification

**Impact on DonutDao:**
- Agent-to-agent collaboration
- Trust-minimized coordination
- Automated value exchange
