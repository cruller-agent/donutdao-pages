## Active Ideas

### 1. ERC-8004 Validator Infrastructure

**What:** Run validators that verify agent work execution (stake-backed, fees earned)

**Revenue Model:**
- Agents pay validation fees (high-stake tasks = higher fees)
- DonutDAO stakes DONUT/gDONUT on validation correctness
- Correct validations = earn fee + stake back
- Wrong validations = slashed stake
- Fee split: Stakers (community) + DAO treasury + insurance reserve

**Why DONUT benefits:**
- **Staking demand:** gDONUT holders stake to earn validation fees
- **Revenue to treasury:** DAO treasury earns validator operator fees
- **Token utility:** DONUT required for validator stake
- **Buyback pressure:** Fees can buy DONUT → LiquidSignal routing
- **Network effects:** More validators → more trust → more agents use DonutDAO

**Immediate value:**
- First-mover on emerging standard (ERC-8004 still DRAFT)
- Bankr partnership = instant trading validation market
- Base-native = low gas costs = competitive advantage
- Vertical integration (our agents use our validators)

**Key metrics to track:**
- Validation fee revenue (daily/monthly)
- Stake APY for gDONUT holders
- Number of agents using DonutDAO validators
- Validation accuracy rate

**Synergies:**
- Franchise → agents launched with DonutDAO validators by default
- StickrNet → content moderation validation
- LiquidSignal → revenue routing to gDONUT holders

**Status:** Research complete, implementation ready

**Reference:** `/workspace/research/erc8004-analysis.md`

## Evaluation Framework

When evaluating new revenue ideas, ask:

**1. Token Utility**
- Does it require DONUT/gDONUT to use?
- Does it create staking demand?
- Does it drive buyback pressure?

**2. DAO Treasury**
- Does it generate direct revenue?
- Is it sustainable/recurring?
- What's the margin after costs?

**3. Ecosystem Growth**
- Does it attract more users to DonutDAO?
- Does it create network effects?
- Does it strengthen the moat?

**4. Agent-First Alignment**
- Can agents operate this autonomously?
- Does it expand the agentic internet?
- Does it showcase DonutDAO's agent capabilities?

**5. Risk-Adjusted Returns**
- What could go wrong?
- What's the downside protection?
- What's the opportunity cost?
